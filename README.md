# hashing
A pure Haskell hash library

## Implemented hash algorithm:

 * SHA1
 * SHA224
 * SHA256
 * SHA384
 * SHA512
 * MD5
 * Whirlpool

## Performance
Performance is about 1/5 to 1/15 of native C/ASM implementation (such as openSSL or cryptonite by Vincent Hanquez).
